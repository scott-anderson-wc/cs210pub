This is an example solution to a new assignment in which data is read from a static .json file and processed in the browser to create a display of all the data. I'm allowing a lot of latitude in the display, and here I've shown one option. A successful solution must do the following:

  • Display all the talks in a compact form. Here, the compact form has the title, authors, start time and location.
  • Clicking on a compact display yields a detailed display that includes the abstract. (You can add other data as well.) Even better, activating it via the space bar or enter key should do the same.
  • Clicking on the detailed display returns to the compact display.
