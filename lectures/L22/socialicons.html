How to create a social icons bar in your page?

The following social icons bar was built using FontAwesome, a font family that includes about 479 icons of most things that can be used as icons in web pages. No images are involed, every icon has a hexadecimal code that is inserted with a special CSS property. You can use CSS to change the color and size of every icon.

This page contains a list of all icons, with the code we need to insert in the CSS file.

The color of the icon is the same for all, in order to match the color scheme. However, you can have every icon with a different color. Click the button to see that happen.

The original colors and icons were chosen with CSS (see source code). The changes are done with jQuery on the fly.
