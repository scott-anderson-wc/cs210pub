A First Exercise

Getting Started

  1. Save this page as "little.html" on your desktop using the File/Save As menu item in Firefox, Safari or IE.

    Be sure to change the format of the saved page

    • In Firefox, change the "Format" to Web Page, HTML only (not Web Page, complete)
    • In IE, change the "Format" option to HTML Source (not Web Archive).
    • In Safari, change the "Format" to Page Source (not Web Archive)
  2. Next, launch TextWrangler. If it's not on your dock, you an click on the Finder (leftmost on the dock), then Go/Applications and scroll down until you see TextWrangler.
  3. Finally, open the file: File/Open and choose "little.html" from the desktop.

Continuing

Edit the file with TextWrangler, save the changes, and view the results in your local web browser just by opening "little.html" (you can even drag "little.html" from your desktop to the browser). There is no need to transfer the file to Puma; you can view the local copy more quickly and easily.

Switch back to TextWrangler, make more changes, save them, switch to your browser, and reload the changes. Repeat as necessary.
