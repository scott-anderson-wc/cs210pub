Photo Gallery: Step-by-step

You can follow along with this example on your own. Start by downloading the folder pg from the cs110dl account. The password for the account is given in the Google Groups. The folder you'll download contains all pictures used for the photo gallery.

The steps below are also described in the lecture slides.

Here is the list of the steps we follow to build the (very simple) photo gallery:

  1. Create a simple HTML page and add the img element to include the first photo from the "panoramas" folder. The result should be the same as in this page.
  2. Use the attribute width to resize the image. This is the result.
  3. Now we can add a second image. This is the result.
  4. Adding all images will show the photo gallery

As we discussed in class, even with the fast Internet connection from Wellesley College, it still takes a few seconds for all images to show in the page. This is why we always are going to resize pictures taken with digital camera, to go from some MB of size to only a few KB of size.
