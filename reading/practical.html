How to Run JavaScript

There are two main ways you will be integrating JavaScript into your HTML code:

  1. Using the <script> tag
  2. Using an external file

This is very similar to CSS, as you can use the <style> tag or link an external CSS file.

If you took CS 111, you used Canopy as your editor. Canopy had a file editor area and a console (also known as a shell) In CS 210, we are using cloud 9 as the editor and the chrome browser for its console, where you can evaluate single JavaScript expressions and test your code for outputs.

To get to the console on a mac:

  1. Open up a chrome browser (any website will do)
  2. On the top bar, go to view --> developer --> JavaScript Console
