Team 1: randomColor()

Your task is to write the body for the function randomColor(). This function will return a color name, for example: red, white, yellow, etc. Remember that color names are strings. Don't use the color blue.. More instructions are in the jsFiddle below.

jsFiddle for randomColor. Write the code in it, test it, then copy the function definition in the shared Google Doc.

Team 2: moveUp()

Your task is to write the body for the function moveUp(). This function, when invoked by the event handler of the UP button, will show the cell moving up on the board. More instructions are in the jsFiddle below.

jsFiddle for moveUp. Write the code in it, test it, then copy the function definition in the shared Google Doc.
