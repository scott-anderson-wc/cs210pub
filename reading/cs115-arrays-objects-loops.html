Arrays, Loops, and Functions

Arrays

Arrays are often used for storing collections of values together under one name. To start off with, here is an empty array with no values.

var emptyArray = [];

We indicate that this is an array by use of the square brackets. This is an array of animals:

var animals = ["cat", "dog", "wallaby"];

Here is a visualization of the same array:

The red numerical indices shown directly in the visualization are implicit when the array is created in code. Note that the index for the first item starts at 0, which is true for all arrays in JavaScript. To get an item from the array, you use the numerical indices. For instance:

var meow = animals[0]; 
var woof = animals[1];
var wallabySound = animals[2];

The variable meow is now "cat", the variable woof is "dog", and the variable wallabySound is "wallaby".

This animals array is homogeneous, as the whole entire array consists of strings. JavaScript arrays can alse be heterogeneous, in which the array consists of values of different types. For example, this array consists of a string, a number, and a boolean: 

var animals = ["kitten", 3.1415, true];

Here are some methods for arrays. Think of the methods as actions performed on the array.

JavaScript Description
array.length returns length of the list/array
array.push(item) Adds item to end.
array.pop() removes the last item and returns the value that was removed.

For example:

var animals = ["cat", "dog", "wallaby"];

animals.length;  // 3
animals.push("panda");  // Adds "panda" to array. Array becomes ["cat", "dog", "wallaby", "panda"]
animals.pop();  // Removes "panda" and returns it. Array becomes ["cat", "dog", "wallaby"]

Objects

Objects are very similar to arrays, in that instead of numerical indices, we can use strings to get an item. We call these strings keys and the items that they get, values.

Here is the code equivalent:

var animals = {kitten:"cat", puppy:"dog", joey:"wallaby"};

The keys and values are separated by a colon.

Keys: Note that the JavaScript keys must be strings, yet kitten, puppy, and joey are not in quotations marks. This is because when you create an object's keys, JavaScript automatically converts them into strings. The quotation marks for keys are optional when writing an object.

Values: Object values can be of any type, such as string, number, boolean, or function. Therefore, you must still use quotation marks, as it specifies that the value is a string as opposed to another type, such as number.

You can get a value by using its key. For instance:

var adultKitten = animal["kitten"];
var adultPuppy = animal["puppy"];
var adultJoey = animal["joy"];

The variable adultKitten is now "cat", adultPuppy is "dog", and adultJoey is "wallaby". Notice that you must still use the quotation marks when indexing with square brackets, even though the original object did not use quotation marks for the keys.

Another way to index values is the following:

var adultKitten = animal.kitten;
var adultPuppy = animal.puppy;
var adultJoey = animal.joy;

This is equivalent to the code above using square brackets, and is called dot notation. When using dot notation, you do not put the keys as strings.

Dot notation is faster to write and clearer to read. However, square bracket notation allows access items when keys have special characters, such as spaces. For example,

var numAnimalsObject = {"number of animals":1000}    // key must use quotations because of spaces 
var numAnimalsValue = numStudentsObject["number of animals"];    // gets item correctly
var numAnimalsValue = numStudentsObject."number of animals";    // error!

To add an item to an object, simply use the same bracket or dot notation for an item that is not yet in the obect.

var zoo = {}    // empty object
zoo.tigers = 2;    // add tigers to zoo object
zoo["panda"] = 1;    // add panda to zoo object
zoo.flamingos = 10    // add flamingos to zoo object
zoo["bears"] = 25;    // add bears to zoo object

In the end, the zoo object will look like so:

var zoo = {tigers:2, panda:1, flamingos:10, bears:25};

Loops

Here is a for-of loop in JavaScript.

var animals = ["cat", "dog", "wallaby"];

// for-of loop
for (var animal of animals) {
  console.log("What a cute " + animal + "!");
}

Let's break this down into a few steps.

  1. We first declare an array of animals, named the variable animals.
  2. for every animal in the array animals
  3. We print the message "What a cute ______! with the blank being an animal.

Therefore, the results of this loop in the console will be:

What a cute cat!
What a cute dog!
What a cute wallaby!

In step two, the variable animal singular can be named anything to your liking. Line 2 could have been for (var fred of animals) { and worked just fine (if you changed line 3 to have fred as well). The important part to remember is that the variable you name is referring to each item in the array, and animals plural is referring to the array itself.

Here is a standard for-loop in JavaScript:

var animals = ["cat", "dog", "wallaby"];

// for loop
for (var i = 0; i < animals.length; i++) {
    console.log("What a cute " + animals[i] + "!";
}

An image to visualize it:

  1. We first create an array of animals.
  2. Assignment: the variable i starts at 0,
  3. Conditional statement: If i is less than the length of animals,
  4. Update: Increment i by one.
  5. Get the element out of the array using the numerical index, and console.log the message.

Let's focus on the last 4 steps. Here is the complete process for this loop:

  • i is assigned to be 0. We check the conditional statement, and 0 is less than 3 (the length of the animals array, as there are 3 items in it). Since the conditional statement is true, "What a cute " + animal[0] "!" is logged, which is "What a cute cat!" i is updated to 1 and the loop continues.
  • i is now 1. We check the conditional statement, and 1 is less than 3. Since the conditional statement is true, "What a cute " + animal[1] "!" is logged, which is "What a cute dog!" i is updated to 2 and the loop continues.
  • i is now 2. We check the conditional statement, and 2 is less than 3. Since the conditional statement is true, "What a cute " + animal[2] "!" is logged, which is "What a cute wallaby!" i is updated to 3 and the loop continues.
  • i is now 3. We check the conditional statement, and 3 is not less than 3. Since the conditional statement is false, the loop stops.

This loop has the same exact results as the for-of loop above, except it uses a numerical index, i, to get its results. You will typically see both kinds of loops in JavaScript.
