Important! Tips on directory and file names:

  • use letters and digits - no special characters like /, *, #, &
  • no spaces - e.g. use cats_and_dogs.html, not cats and dogs.html
  • case matters - cats.html not the same as Cats.html

Directory Trees

Relative URLs

  • Files on the same server - don't need protocol and server name
  • File in the same folder: link in buffy.html to the file willow.html

      <a href="willow.html">her best friend Willow</a>

  • File in a sibling folder: link in mythology.html to the file angel.html

      <a href="cast/angel.html">Angel, the vampire with a soul</a>

  • File in a folder "above": link in host.html to the file season3.html

      <a href="../season3.html">the Host returned in season 3</a>

  • File in a folder that's multiple levels up: link in buffy.html to the file comics.html

      <a href="../../comics.html">another layer in the comics</a>

  • Add a link in comics.html to the file gunn.html
  • Add a link in host.html to the file buffy.html
